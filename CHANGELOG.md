# Changelog

All notable changes to the "Geant4 Text Geometry" extension will be documented in this file.

## [1.0.1] - 2026-02-10

### Fixed

- Fixed empty "Changes" tab on Open VSX by adding this changelog.

### Added

- **Contributor Tooling**: Added `CONTRIBUTING.md` to guide community contributions.
- **Issue & PR Templates**: Standardized templates for bug reports, feature and pull requests.
- **Developer Experience**: Integrated Prettier for automatic code formatting to ensure consistent style.

## [1.0.0] - 2026-01-25

- Initial release.
- Syntax highlighting for `.tg` files.
- Snippets for Geant4 solids, volumes, materials, and more.
